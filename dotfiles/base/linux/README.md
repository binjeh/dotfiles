# Base Dot Files

Base configuration files installers for Linux.

## Script Purpose

* `00_configure_bash.sh`
  * **REQUIRED**
  * Configures bash for base usage.

* `01_configure_tmux.sh`
  * **OPTIONAL**
  * Configures TMUX for usage.

* `02_configure_neovim.sh`
  * **OPTIONAL**
  * Configures neovim for usage.
